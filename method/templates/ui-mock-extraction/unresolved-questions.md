4. <Card / CTA / Chip / Badge / IconButton などは既存デザインシステム部品か、新規 E-DESIGN-* として構築するか。>
5. <型チップ、条件チップ、候補値チップ、範囲表示の色・意味対応は K-DESIGN として固定するか。>
